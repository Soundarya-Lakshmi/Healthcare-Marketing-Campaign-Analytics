# Overview

This project simulates healthcare marketing analytics to evaluate how campaigns drive patient engagement and conversion.
The solution focuses on analyzing campaign performance, understanding patient behavior, and enabling data-driven decision-making through interactive dashboards.

# Dashboard Highlights

1. Campaign Performance Overview
- KPI tracking: Patients Reached, Engagement %, Conversion %, Revenue, Profit %
- Patient-level conversion funnel
- Campaign type performance comparison

2. Patient Insights & Engagement Analysis
- Engagement and conversion by medical condition
- Channel-wise conversion analysis
- Patient engagement frequency distribution

3. Hospital Performance & Benchmarking
- Dynamic conversion threshold filtering
- Identification of underperforming hospitals
- Comparative performance analysis

  ## Impact

This project demonstrates how marketing analytics can be applied in healthcare to optimize campaign strategies, improve patient engagement, and support data-driven decisions.
